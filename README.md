## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/achrafdev89/ai-image-generator.git
```

### Install backend dependencies

```bash
cd backend
npm install
```

### Configure environment variables

```env
OPENAI_API_KEY=your_api_key
PORT=5000
```

### Start backend

```bash
npm run dev
```

### Start frontend

Open:

```text
frontend/index.html
```

or use VS Code Live Server.
